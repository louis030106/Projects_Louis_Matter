# Lap Timing System for Miniature Racing Cars

## Objective
Create an automated system capable of detecting miniature cars on a racetrack and calculating their performance metrics in real time.

## Problem
Manual lap timing is inaccurate and does not provide consistent or detailed performance data such as average speed or best lap.

## Developed Solution
I built a system that:
- detects each car as it completes a lap,
- measures lap time with high precision,
- calculates average lap speed,
- records the best lap,
- displays all information continuously on a screen.

## Results
- Accurate and real-time lap measurements.
- Clear and permanent on-screen display.
- Easy performance comparison between laps.
- Suitable for competitions or repeated testing.

## Skills Demonstrated
- Sensor signal processing  
- Real-time computation  
- Embedded display programming  
- Performance data analysis  
- End-to-end system design (detection → processing → display)
